# 📌 Objectives

To achieve our ultimate goal, the SentinelAI project meets the following specific objectives:

1.  **Immersive 3D Panel Simulation:**
    Build a virtual boardroom environment featuring three specialized AI interviewers, each with dynamic facial animations and unique responsibilities (The Architect for code, The Manager for speech clarity, The Observer for behavior).

2.  **Real-Time Biometric Analysis:**
    Use standard computer vision (via a simple webcam) to measure eye blink rates, detect posture stability, count gaze deviations, and trace emotional micro-expressions in real-time.

3.  **Adaptive Difficulty & Interactive Mechanics:**
    Build a dynamic engine that automatically increases the difficulty of coding questions when the candidate answers brilliantly, or softens them if they struggle. Program AI agents to politely interrupt if the candidate rambles.

4.  **Privacy-First Customization:**
    Give users the choice to run the entire interview pipeline offline (locally using an Ollama daemon) with custom courtroom personalities (Brutal, Coaching, or Academic) to guarantee absolute data privacy.

5.  **Actionable & Shareable Analytics:**
    Deliver a comprehensive performance scorecard complete with historical progression benchmarks, customized growth plans, and unauthenticated public shareable links.
